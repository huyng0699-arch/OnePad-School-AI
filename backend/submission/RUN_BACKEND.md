# Run Backend

Runtime required:
- Node.js LTS
- npm

Not required:
- Docker
- Python
- Supabase local

Run:
- `scripts\setup-backend.bat`
- `scripts\start-backend.bat`

Console URL:
- `http://localhost:3000/index.html`

Pipeline test:
- `powershell -ExecutionPolicy Bypass -File scripts\test-pipeline.ps1`
